# پشتیبان هوشمند آریا - WordPress Chatbot Plugin

پلاگین چت بات هوشمند آریا برای وردپرس با قابلیت‌های پیشرفته صوتی، حافظه، شخصیت‌پردازی و یکپارچگی کامل.


نمای کلی پروژه
یک پلاگین هوشمند WordPress که با استفاده از OpenAI API، چت بات پیشرفته‌ای برای سایت‌های مختلف ایجاد می‌کند. این پلاگین قابلیت تطبیق هوشمند با انواع مختلف وب‌سایت‌ها دارد و به‌صورت کاملاً خودکار اطلاعات سایت را تحلیل و در پاسخ‌هایش استفاده می‌کند.



## ویژگی‌ها

- پشتیبانی از گفتار به متن و متن به گفتار
- سیستم حافظه پیشرفته برای یادآوری مکالمات قبلی
- موتور شخصیت‌پردازی هوشمند
- یکپارچگی با سیستم‌های مدیریت محتوا
- رابط کاربری زیبا و واکنش‌گرا
- پشتیبانی از چند زبان
- قابلیت شخصی‌سازی پیشرفته

## نیازمندی‌ها

- PHP نسخه 7.4 یا بالاتر
- وردپرس نسخه 5.0 یا بالاتر
- پسوند cURL
- پسوند JSON

## نصب

1. فایل پلاگین را در پوشه `/wp-content/plugins/` آپلود کنید
2. پلاگین را از طریق منوی 'افزونه‌ها' در وردپرس فعال کنید
3. به بخش تنظیمات پلاگین بروید و تنظیمات مورد نظر را انجام دهید

## 🎤 قابلیت‌های صوتی پیشرفته (Voice Features)

### Speech Recognition (تشخیص گفتار)
- **ورودی صوتی**: امکان صحبت با چت بات به جای تایپ
- **تشخیص زبان خودکار**: شناسایی خودکار زبان صحبت کاربر  
- **فیلتر نویز**: حذف صداهای پس‌زمینه
- **دقت بالا**: استفاده از Web Speech API برای دقت بالا
- **Real-time Processing**: پردازش لحظه‌ای صدا

### Text-to-Speech (سنتز گفتار)
- **تبدیل متن به گفتار**: خواندن پاسخ‌های چت بات
- **بیش از 40 زبان**: پشتیبانی از زبان‌های مختلف
- **بیش از 160 صدا**: انتخاب صدای مردانه و زنانه
- **کنترل سرعت**: تنظیم سرعت صحبت
- **کنترل تن صدا**: تنظیم pitch و tone
- **Natural Voice**: صداهای طبیعی و انسان‌وار

### Sound Effects (جلوه‌های صوتی)
- **صدای ارسال پیام**: بازخورد شنیداری برای ارسال
- **صدای دریافت پیام**: اعلان صوتی برای پاسخ‌های جدید
- **صداهای سفارشی**: آپلود صداهای شخصی
- **Volume Control**: کنترل صدا توسط کاربر
- **Mute Option**: امکان قطع کامل صدا

## 💾 مدیریت حافظه و یادگیری (Memory & Learning)

### Conversation Memory
- **ذخیره مکالمات**: نگهداری تاریخچه مکالمات
- **Session Continuity**: ادامه مکالمه در بازدیدهای بعدی
- **User Profiling**: ایجاد پروفایل کاربری بر اساس مکالمات
- **Context Retention**: حفظ زمینه مکالمه
- **Smart Reminders**: یادآوری موضوعات قبلی

### Adaptive Learning
- **یادگیری تطبیقی**: بهبود پاسخ‌ها بر اساس بازخورد
- **Pattern Recognition**: شناسایی الگوهای سؤال
- **Response Optimization**: بهینه‌سازی پاسخ‌ها
- **User Preference Learning**: یادگیری ترجیحات کاربر
- **Feedback Integration**: ادغام بازخوردهای کاربران
